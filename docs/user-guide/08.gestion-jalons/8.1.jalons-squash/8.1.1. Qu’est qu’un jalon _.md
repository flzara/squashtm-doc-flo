# Qu’est qu’un jalon ?

3 encarts différents :

 - Focus : remarques à mettre en avant 
 - Attention : Evoquer un point de vigilance 
   En savoir plus : Faire un renvoi vers une autre page

> Votons le nom de l'encart :  Focus ? Note ? Info ?
